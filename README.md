# Realmart Sales Analytics 📊

## Overview
A comprehensive Power BI dashboard analyzing the **Realmart Sales Dataset** to uncover key retail performance metrics. This project transforms raw sales data into actionable business insights, focusing on revenue trends, product category performance, and regional sales distribution.

## Key Features
- **Interactive Dashboards:** Visualizations of monthly and quarterly sales growth.
- **Product Analysis:** Identification of high-performing product categories and sub-categories.
- **Regional Insights:** Sales breakdown by region for better resource allocation and market strategy.
- **Data-Driven KPIs:** Tracking of total revenue, profit margins, and sales volume.

## Tools & Technologies
- **Power BI:** Data modeling, DAX calculations, and interactive report design.
- **Microsoft Excel:** Source dataset management and data cleaning.

## Files in this Repository
- `ccbp-power-BI.pbix`: The main Power BI project file.
- `Realmart_Sales_Dataset+.xlsx`: The raw dataset used for analysis.

## How to View
1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
2. Clone this repository or download the `.pbix` file.
3. Open `ccbp-power-BI.pbix` in Power BI Desktop to explore the dashboard.
